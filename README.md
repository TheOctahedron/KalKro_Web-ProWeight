## REQUIREMENTS & LAUNCH
REQUIRED LIBRARIES (Python):

1. **fastapi**

2. **uvicorn**

**VERSION**: Python **3.7+**

**1. Install the repository.** (git)
```bash
git clone https://github.com/TheOctahedron/KalKro_Web.git
cd KalKro_Web
```

**2. Install Backend.**
```bash
cd backend
pip install -r requirements.txt 
```
OR:

```bash
cd backend
pip install fastapi uvicorn
```

**3. Install Frontend.** (Always in the NEW terminal.)
```bash
cd ../frontend
npm install
```
---
**If everything is successful**, here's what to paste into the browser: 

**For the backend:**
```bash
http://localhost:8000
```

**For the frontend:**
```bash
http://localhost:5173
```

**For the backend docs:**
```bash
http://localhost:8000/docs
```
